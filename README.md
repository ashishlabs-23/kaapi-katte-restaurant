A premium, high-aesthetic web application for **Kaapi Katte**, a South Indian restaurant that blends traditional heritage with a modern, minimalist digital experience.

## ✨ Features

*   **Modern Temple Aesthetic**: Custom glassmorphism design system inspired by South Indian architecture.
*   **Intuitive Ordering**: Staggered menu grid with real-time cart management.
*   **Staff Kitchen Portal**: Live order manifest for real-time status updates.
*   **Owner's Intelligence Lounge**: Secure dashboard with business metrics, profit analysis, and demo mode.
*   **Seamless Payments**: Integrated with Google Apps Script for lightweight, spreadsheet-based order logging.
*   **Fully Containerized**: Ready for development and production via Docker.

## 🛠 Tech Stack

*   **Frontend**: React (Vite)
*   **Logic**: React Router, Context API
*   **Styling**: Vanilla CSS (Modern CSS Variables & Tokens)
*   **Icons**: Lucide React
*   **Infrastructure**: Docker, Docker Compose
*   **Backend Sync**: Google Apps Script & Google Sheets
*   **Automation**: CodeRabbit AI Reviews

## 🚀 Getting Started

### Using Docker (Recommended)

1.  Clone the repository.
2.  Run the following command:
    ```bash
    docker-compose up --build
    ```
3.  Open [http://localhost:5173](http://localhost:5173).

### Local Development

1.  Install dependencies:
    ```bash
    npm install
    ```
2.  Start the development server:
    ```bash
    npm run dev
    ```

## 🔐 Administrative Access

*   **Kitchen Manifest**: Accessible via `/kitchen`.
*   **Owner's Dashboard**: Accessible via `/dashboard` (Direct access PIN required).
